# Legacy Android (Java)

Исходное нативное приложение **Time2T3YourPills** (Single Activity, Room, TimerService). Сохранено для справки при миграции на Flutter.

Сборка из этой папки:

```bash
cd legacy_android
./gradlew assembleDebug   # Linux/macOS
gradlew.bat assembleDebug # Windows
```

Актуальная разработка ведётся в корне репозитория: Flutter (`lib/`, `pubspec.yaml`).
